# EJLive Project Memory

## Project identity

EJLive Enterprise is a C#/.NET WinForms-based ATM electronic journal archiving, synchronization, monitoring, reporting, and controlled operations system.

## Core domains

- ATM electronic journal capture and transfer
- client/server synchronization
- monitoring dashboard / NOC views
- server-side analytics and reporting
- vendor-specific ATM log analysis
- XFS/NDC event mapping
- archive indexing
- safe Windows service integration
- controlled remote-support workflows
- installer/deployment wiring

## Vendor scope

Supported or planned ATM families:
- NCR
- GRG
- Wincor/Nixdorf / Diebold-Nixdorf

## Active architecture expectation

Treat the active project structure as the authority. Legacy/original and reference documents are sources for missing capabilities, not files to compile blindly.

## Key project rule

Do not lose legacy capabilities, but do not break the current build by compiling raw legacy files with duplicate names or old dependencies.

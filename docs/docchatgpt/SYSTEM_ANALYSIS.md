## 🎯 المتطلبات الشاملة للنظام

### 1. الواجهات الرئيسية (3 تطبيقات)

#### أ) تطبيق العميل (Client) - على أجهزة الصراف
**التبويبات:**
1. **Dashboard**
   - حالة الاتصال بالخادم
   - آخر مزامنة (التاريخ والوقت)
   - عدد ملفات الجورنال المعلقة
   - حالة التشفير والضغط
   - إحصائيات المعاملات

2. **Journal Sync**
   - حالة المزامنة (جارية/متوقفة/منتهية)
   - شريط تقدم المزامنة
   - سجل المزامنات السابقة
   - عدد الملفات المرسلة
   - حجم البيانات المرسلة

3. **Remote Control**
   - إعادة تشغيل الجهاز
   - التقاط لقطة شاشة
   - مزامنة الوقت مع الخادم
   - إيقاف/بدء المزامنة

4. **Settings**
   - نوع الصراف (NCR/GRG/WN)
   - عنوان IP الخادم
   - منفذ الاتصال
   - مسار ملفات الجورنال
   - فترة المزامنة
   - مفتاح التشفير (محمي)

5. **Logs**
   - سجل الأحداث
   - سجل الأخطاء
   - سجل الاتصالات
   - تصفية حسب النوع والتاريخ

#### ب) تطبيق الخادم (Server) - على الخادم المركزي
**التبويبات:**
1. **Server**
   - حالة الخادم (تشغيل/إيقاف)
   - منفذ الاستماع
   - مسار التخزين
   - عدد الاتصالات النشطة
   - زر Start/Stop Server

2. **Connected ATMs**
   - قائمة الصرافات المتصلة
   - معلومات كل صراف: ID, IP, Type, Last Sync
   - حالة الاتصال (أخضر/أحمر)
   - آخر وقت اتصال
   - عدد الملفات المستقبلة

3. **Journal Repository**
   - استعراض ملفات الجورنال المستقبلة
   - البحث حسب ATM ID أو التاريخ
   - عرض محتوى الملف
   - تصدير الملفات
   - حذف الملفات القديمة

4. **Archive Manager**
   - إدارة الأرشفة الشهرية
   - ضغط الملفات القديمة
   - نقل الملفات إلى التخزين الثانوي
   - استرجاع الملفات المؤرشفة
   - إحصائيات الأرشفة

5. **Remote Commands**
   - اختيار صراف من القائمة
   - إرسال أوامر: Restart, Screenshot, Sync Time
   - عرض نتائج الأوامر
   - سجل الأوامر المرسلة

6. **Reports & Analytics**
   - تقارير المزامنة اليومية
   - إحصائيات الأخطاء
   - رسوم بيانية للنشاط
   - تقارير الأداء
   - تصدير التقارير (PDF/Excel)

7. **Settings**
   - إعدادات الخادم (منفذ، مسار التخزين)
   - إعدادات الأمان (مفتاح التشفير)
   - إعدادات النسخ الاحتياطي
   - إعدادات البريد الإلكتروني للتنبيهات

8. **Logs**
   - سجل الأحداث الشامل
   - سجل الأخطاء
   - سجل الاتصالات
   - تصفية وبحث متقدم

#### ج) لوحة المراقبة (Monitoring Dashboard)
**التبويبات:**
1. **Dashboard**
   - إحصائيات رئيسية: Total ATMs, Online, Synced, Faults
   - أزرار الإجراءات السريعة
   - تنبيهات حية

2. **ATM Network**
   - خريطة بصرية للصرافات
   - حالة كل صراف (أخضر/أحمر/أصفر)
   - معلومات عند الضغط على الصراف

3. **Alerts**
   - قائمة التنبيهات
   - تصفية حسب الأولوية والنوع
   - وضع علامة على التنبيهات كمقروءة

4. **Archive**
   - استعراض الملفات المؤرشفة
   - البحث والتصفية
   - تحميل الملفات

5. **Settings**
   - إعدادات المراقبة
   - إعدادات التنبيهات

## 🔐 متطلبات الأمان والاتصال

### التشفير
- **AES-256:** مفتاح 32 بايت، IV 16 بايت
- **تشفير جميع البيانات المنقولة** عبر الشبكة
- **مفتاح موحد** لجميع الصرافات والخادم

### الاتصال
- **TCP/IP:** اتصال آمن على المنفذ 5005
- **Heartbeat:** فحص الاتصال كل 30 ثانية
- **Reconnection Logic:** إعادة محاولة الاتصال تلقائياً
- **Timeout:** انقطاع الاتصال بعد 5 دقائق من عدم الاستجابة

### التحكم عن بعد
- **Remote Desktop:** عرض شاشة الصراف
- **Remote Commands:** إعادة تشغيل، التقاط شاشة، مزامنة وقت
- **Authentication:** التحقق من الهوية قبل تنفيذ الأوامر

## 📊 متطلبات مزامنة الجورنال

### معالجة الملفات
- **NCR:** ملفات جورنال ثابتة (Fixed)
- **GRG:** ملفات جورنال يومية
- **WN (Wincor Nixdorf):** ملفات جورنال يومية مع دعم TRACE

### المزامنة
- **استقلالية عالية:** تعمل المزامنة حتى لو انقطع الاتصال
- **تخزين محلي:** حفظ الملفات المعلقة محلياً
- **إعادة محاولة:** إرسال الملفات المعلقة تلقائياً عند استعادة الاتصال

### التخزين على الخادم
- **مجلد منفصل لكل صراف:** `/Storage/ATM_{ID}/`
- **تنظيم حسب التاريخ:** `/Storage/ATM_{ID}/2026/05/`
- **ملفات معلومات:** `metadata.json` لكل ملف جورنال

### التحليل
- **عداد المعاملات:** عدد المعاملات في كل ملف
- **الأخطاء:** تحديد الأخطاء في الجورنال
- **الإحصائيات:** حساب الإحصائيات من الجورنال
